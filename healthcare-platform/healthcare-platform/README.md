# Healthcare Booking Platform

Full-stack healthcare appointment booking platform.

## Stack
- React
- Node.js
- Express
- PostgreSQL
- Docker

## Features
- JWT Authentication
- Doctor Search
- Appointment Booking
- Role-Based Access
- PostgreSQL Integration

## Run Backend

```bash
cd backend
npm install
npm run dev
```

## Run Frontend

```bash
cd frontend
npm install
npm start
```

## PostgreSQL Setup

```sql
CREATE DATABASE healthcare_booking;
```

## Docker

```bash
docker-compose up --build
```
